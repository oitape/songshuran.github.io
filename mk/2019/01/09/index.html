MK简单使用 [查看更多]

发布 : 2019-01-09 分类 : 技术 浏览 :

###markdown通过简单的的标记语法就可以使自己的文本内容更加（you）美（bi）观（ge），而不需要掌握太多网页的知识，方便我们使用，所以掌握markdown语法还是很有必要的，下面就简单的介绍下markdown快速入门的语法要点。

标题使用 H

这是H1

这是h2

这是h3

这是h4

这是h5
这是h6

#####行首插入 1 到 6 个 # ，对应到标题 1 到 6 阶。行首的井字符数量决定标题的阶数。 行尾的#可以不写，但为了美观可以加上 ###

区块引用 Blockquotes

This is a blockquote with two paragraphs.

在行首用>来作为区块引用的开始

换行符

使用html中的<br> 作为换行符，在恰当的地方对某一段长文字进行换行

列表

Markdown 支持有序列表和无序列表。无序列表使用*、+或是-作为列表标记，这三种符号都可达到相同的效果；有序列表则使用数字接着一个英文句点.。

  • 红色
  • 绿色
  • 黑色
  1. 男人
  2. 女人
  3. 其他
  • ios
  • android

如果在行首出现数字-句点-空白时，避免列表形式展示，可以在句点前面加上反斜杠 \

111.2 What a great season.

代码区块

代码块展示（需要四个空格键，或者两次tab键） 

谷歌     谷歌     谷歌     谷歌     谷歌   谷歌   谷歌

分割线

产生分割线需要用*, - 来作为分割线（不少于三个连续的符号，中间可以有空格）

区块元素

链接

Markdown 支持两种形式的链接语法： 行内式和参考式两种形式。不管是哪一种，链接文字都是用 [方括号] 来标记。

行内式

This is an example inline link.

This is [an example](http://www.baidu.com/ “Title”) inline link.

参考式

This is an example reference-style link.

This is [an example] [id] reference-style link.
[id]: http://www.baidu.com/ “Optional Title Here”

隐式连接（可以省略指定链接标记）

google

[google][]
[google]: http://www.google.com/

强调

Markdown 使用星号*和底线_作为标记强调字词的符号，如果 * 和 _ 两边都有空白的话，它们就只会被当成普通的符号。
使用** 或者 __ 可以使“强调”字体加粗。

fishcoder

我叫冉冉，一个大帅锅。嘿嘿……

代码

可以用反引号把它包起来（`）

例如：｀｀｀中间放入代码片段｀｀｀

如果要在代码区段内插入反引号，用多个反引号`来开启和结束代码区段：

一个惊叹号 !
接着一个方括号，里面放上图片的替代文字
接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上 选择性的 ‘title’ 文字。
```

其他

自动连接

Markdown 支持以比较简短的自动链接形式来处理网址和电子邮件信箱，只要是用方括号包起来， Markdown 就会自动把它转成链接。一般网址的链接文字就和链接地址一样.

http://www.baidu.com

反斜杠

Markdown 可以利用反斜杠来插入一些在语法中有其它意义的符号
